# Santi Younger Tools

Install and update **Santi Younger's** plugins and themes with a simple platform login on [platform.santiyounger.com](https://platform.santiyounger.com).

Paid items need a platform account.

## Install

### Community plugins

1. Open **Settings → Community plugins**.
2. Turn off **Restricted mode** if it is on.
3. Select **Browse**, search **Santi Younger Tools**, then **Install** and enable.
4. Reload Obsidian when prompted.

### BRAT (before directory listing)

**Santi Younger Tools** is not in the community plugin directory yet. Install it with **[BRAT](https://github.com/TfTHacker/obsidian42-brat)** (Beta Reviewers Auto-update Tester)—a community plugin that installs and updates other plugins from a GitHub repository.

#### Step 1 — Install BRAT

1. Open **Settings → Community plugins**.
2. Turn off **Restricted mode** if it is on.
3. Select **Browse**, search for **BRAT**, install **Beta Reviewers Auto-update Tester**, then enable it.

#### Step 2 — Add this plugin through BRAT

1. Open **BRAT** settings.
2. Scroll down to **Beta plugin list**.
3. Select **Add beta plugin**.
4. Paste: `https://github.com/santiyounger/obsidian-santi-younger-tools`
5. Select **Add plugin**, then enable **Santi Younger Tools** and reload Obsidian when prompted.

## Use

1. Ribbon **package** icon or command **Open tools**.
2. **My account** → sign in.
3. **Plugins** or **Themes** → **Install** (reload if prompted).

When signed in, catalog plugins auto-update shortly after Obsidian loads; use **Check for updates** in the panel anytime. Hub updates come from **Community plugins** when listed, or through **BRAT** until then.
