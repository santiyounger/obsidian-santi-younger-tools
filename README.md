# Santi Younger Tools

Install and update **Santi Younger's** plugins and themes with a platform login on [platform.santiyounger.com](https://platform.santiyounger.com). For Santi Younger clients. Sign in with the email you use for purchases to access the catalog.

The catalog is limited to Santi Younger products.

## Disclosure

| Topic | Detail |
| --- | --- |
| Network | `platform.santiyounger.com` (login, entitlements, install files). Free catalog entries may use public GitHub releases. Preview images from `santiyounger.b-cdn.net`. No telemetry. |
| Data | Session and install metadata in plugin `data.json`. Writes `.obsidian/plugins/` and themes on install. |
| Commercial | Paid items require platform access. Purchases on the website, not in-plugin. |
| Account | Sign-in required to install and update catalog items. Use the email from your Santi Younger purchase or account. |
| Other plugins | Installs separate Santi catalog products into the vault. Many are not in the community directory and have their own terms. |

## Install

Install this plugin with **[BRAT](https://github.com/TfTHacker/obsidian42-brat)** (Beta Reviewers Auto-update Tester), a community plugin that installs and updates other plugins from a GitHub repository.

### Step 1: Install BRAT

1. Open **Settings → Community plugins**.
2. Turn off **Restricted mode** if it is on.
3. Select **Browse**, search for **BRAT**, install **Beta Reviewers Auto-update Tester**, then enable it.

### Step 2: Add this plugin through BRAT

1. Open **BRAT** settings.
2. Scroll down to **Beta plugin list**.
3. Select **Add beta plugin**.
4. Paste: `https://github.com/santiyounger/obsidian-santi-younger-tools`
5. Select **Add plugin**, then enable **Santi Younger Tools** and reload Obsidian when prompted.

## Use

1. Run **Open tools** (ribbon **package** icon or command palette).
2. Enter your platform email and select **Send login code**. If you are already a client, you will receive a 6-digit code by email.
3. Enter the code to sign in, then open **Plugins** or **Themes** and select **Install** (reload if prompted).

When signed in, catalog plugins auto-update shortly after Obsidian loads. Use **Check for updates** in the panel anytime.
